# riyu
## 文字分类
汉字 ：中国汉字，但是意思不一样，发音也不一样
平假名：由汉字的古文平易过来
片假名：由汉字的 偏旁部首或者一半演化过来
罗马字：一般用于名称等。。。
## 语音特点
1.开音节
每个语音以元音结尾
2.等时性
每个音节一拍，发音长短基本相同

## 概念
声调：高低型
轻音与浊音：辅音发音时声带的震动与否   例如：k g
